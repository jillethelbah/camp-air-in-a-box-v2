## 🏗️ How This Was Built

This entire site was built using **GitHub Copilot** in an iterative, conversational workflow:

1. Started with a single-page concept for delivery model selection
2. Iterated through feedback from the Camp AIR squad (Dean, Aparna, Thomas, Ronit, Jared)
3. Added interactive features: drag-and-drop agenda builder, audience picker, expandable panels
4. Split into multiple pages as scope grew (Administration, Guide)
5. Incorporated meeting transcript insights (Jill & Kyran 1:1) for the support model

**Tech stack:** Single-file HTML pages with embedded CSS/JS. No build tools. Microsoft Fluent-inspired design system. Hosted on GitHub Pages.

## 🎯 What's Inside

- **3 Delivery Models**: Manager-Led, V-Team-Led (Org/Division), Scale-from-Within
- **Agenda Builder**: Standard 3-week agenda + drag-and-drop custom builder
- **5-Day Immersion Schedule**: Full session breakdown with recordings & practice links
- **2-Tier Support Model**: Self-Service + Advisory (free through FY27 H1)
- **Interactive Guide**: 5-phase playbook with persistent checklist
- **Feedback Loop**: Jul–Dec tracking framework with 6 metric areas
